#### 커밋 메세지 형식

```
[type] 구체적인 내용
```
- 새기능 추가: [feature] 검색 기능
- 버그 수정: [fix]
- 디자인 수정(css) : [design]
- 코드 스타일(포맷팅 등) 변경: [style]
- 리팩토링: [refactor]
- 기타 등등: [chore]
